# Day 3
Solution to [Day 3](https://adventofcode.com/2023/day/3).


## Requirements
    * CMake >= 3.22


## Linux execution

```
git clone git@github.com:SalaheddineSTA/Advent-Of-Code-2023.git
cd Day03
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release
cd build & make -j8
./AdventCode ../Data/input.txt
```
